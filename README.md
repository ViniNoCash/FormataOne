# FormataOne

Aplicação desktop em Java (Swing) para formatação de CPF e CNPJ, com interface gráfica, cópia para área de transferência e limpeza de campos.

## 🚀 Tecnologias

- Java
- Swing (GUI)

## ⚙️ Como rodar

### Opção 1 — Executável (mais fácil)
Baixe o arquivo `FormataOne-1.0.exe` e execute diretamente, sem precisar instalar nada.

### Opção 2 — Pelo código fonte
1. Clone o repositório:
```bash
   git clone https://github.com/ViniNoCash/FormataOne.git
```
2. Abra o projeto no IntelliJ IDEA
3. Execute o arquivo `FormatadorCash.java` localizado em `src/`

## 📋 Funcionalidades

- Formata CPF e CNPJ removendo caracteres não numéricos
- Copia o resultado para a área de transferência
- Limpa os campos com um clique
